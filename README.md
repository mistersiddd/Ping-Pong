# Ping-Pong

A simple multiplayer Ping Pong game developed using python

## To run and play

```
git clone https://github.com/mistersiddd/Ping-Pong.git
cd Ping-Pong
python3 pingpong.py
```

Player A has 'W' and 'S' for moving the left paddle and Player B has 'Up' and 'Down' arrows for moving the right paddle
